Qu'est-ce que SHT ?

SHT est un projet devellope par Jpinx dans le but de former gratuitement des informaticiens debutants (et bientot intermediaires) dans le domaine de la cybersecurite.Il s'agit d'un programme contenant des cours, des utilitaires et bienot un simulateur de test.
Les cours sont encore en devellopement et incomplets.

  • Des la version 3.0.4, un simulateur de pentest hors-ligne avec cibles personalisables !!

Installation de SHT

Pour installer n'importe quelle version de SHT, assurez vous d'avoir les configurations minimales , puis rendez-vous sur pearltrees
La version 2.0.3 est fortement recommandee pour une utilisation securisee,stable etlong-terme.

  1. Entrez dans le dossier informatique&programmation
  2. Rendez-vous dans la section Simulateur Hacking Toolkit
  3. Telechargez la derniere version (soit fichier .zip soit tous les fichiers)
  4. Verifiez que l'emplacement des fichiers est bien correct les txt(ou sht) sont dans le fichier txt(ou sht) pour la version 2.0.1 et 2.0.2
  5. Pour les versions optimisees 2.0.1, 2.0.2 et 2.0.3 assurez-vous que la valeur de compte.txt est négative il doit contenir "0" au depart
  6. Executez SHT.exe

Importance de ces fichiers

Les fichiers sht contiennent soit les cours, soit des donnees personnelles, les cours ont ete cryptes e maniere a ne pas pouvoir etre recuperer facilement.
Les fichiers .o permettent l'execution integrale de sht. Les .dll sont relatifs a l'application sht et L'excutable est le programme en lui-meme
Les fichiers .o et .dll sont indispensables

Les erreurs

En executant le programme, vous pouvez rencontrer des erreurs

Les erreurs dont nous connaissons l'origine et dont nous pouvons vous regler commencent par 00000xQ
Le nombre qui suit permet d'identifier l'erreur
Avec la version 2.3.1, la commande maintenace permet de lister les erreurs connus et de les identifier

  • Si vous avez l'erreur 00000xQ1, il vous manque le fichier compte.txt Telechargez le ICI
  • Si lors du demarrage,un message d'erreur s'affiche et indique qu'il vous manque un libxxxxx.dll, vous pouvez trouvez ce fichier sur internet ou pearltrees de jpinx
  • Si vous avez l'erreur 00000xQ4, il vous manque le(s) fichier(s) coursX.txt telechargez sur le site pearltrees de jpinx
  • Si vous avez l'erreur 00000xQ3, il manque le fichier presentation.sht
  • Si vous avez l'erreur 00000xQ5 ou 2, il manque le fichier pseudo.sht
  • Si vous avez l'erreur 00000xQ6, il vous manque le fichier dico.sht
  • Si vous avez l'erreur 00000xQ7, il vous manque le fichier m.sht
